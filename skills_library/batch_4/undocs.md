Summary

Zero-config documentation site generator built on Nuxt, Nuxt Content, and Nuxt UI.

Includes CLI commands for development and production builds with automatic content transformations
Provides built-in markdown components for alerts, package manager commands, diagrams, and cross-page links
Supports blog posts, SEO optimization with Open Graph images, and raw markdown output for LLM integration
Configurable via docs.yaml with customizable navigation, landing pages, icon collections, and content collections
SKILL.md

Undocs is a minimal Documentation Theme and CLI built with Nuxt, Nuxt Content, and Nuxt UI. It provides zero-config documentation sites with elegant CLI wrapper, automatic content transformations, and built-in components for common documentation needs.

The skill is based on undocs v0.4.15, generated at 2026-01-31.
