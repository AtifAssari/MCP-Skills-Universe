Python Engineering Skill Policy

Code is Mechanism, Prompt is Policy

Python Standards (v2.0 - Modern)

When writing or editing Python code:

State Management - Use StrEnum instead of magic strings
Control Flow - Prefer match/case over complex if/elif chains
Type Safety - Use @override for inherited methods and new generic syntax class Foo[T]
Concurrency - Use asyncio.TaskGroup instead of gather
Standards - Follow PEP 8, 4 spaces, and Google style docstrings

See lang-python-modern.md for complete standards.
