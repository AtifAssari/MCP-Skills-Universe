# plot-keypoints

SKILL.md
大情节点分析专家
功能

梳理故事主线，提炼并按故事发展阶段排列主要情节点。

使用场景
快速掌握故事的整体结构和情节发展。
为详细情节分析提供基础框架。
制作故事大纲或梗概。
剧本改编前进行结构梳理。
核心步骤
阅读理解: 充分阅读并理解故事文本，把握整体结构和主题。
脉络梳理: 梳理故事准确完整的脉络，识别关键转折点。
情节点提炼: 根据情节点定义，准确总结主要情节点，确保完整准确。
阶段划分: 按故事发展阶段（如三幕式结构）进行划分和排列。
结构化输出: 按照指定格式输出，确保清晰易读。
输入要求
完整的故事文本（小说、剧本、故事大纲等）
建议文本长度：1000字以上以获得更完整的分析
输出格式
【阶段一：情节主旨】：
- 子情节点1
- 子情节点2
- 子情节点3

【阶段二：情节主旨】：
- 子情节点1
- 子情节点2
- 子情节点3

【阶段三：情节主旨】：
...

约束条件
每个情节点表述不超过150字。
严格按照故事文本原文意思总结，不自行创作改编。
确保主要情节点的总结完整、准确、细致。
不使用阿拉伯数字为情节点标号。
