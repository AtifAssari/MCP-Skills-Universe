About Tella
Why TellaTV Downloader

Tella makes it easy to publish polished screen recordings and presentations, but viewers do not always get a simple built-in download option. Videos are served through streaming playback, which means right-click save workflows do not help, and generic downloader tools often fail to detect Tella embeds reliably.

TellaTV Downloader is built specifically for that workflow. It focuses on Tella pages and supported embedded players, detects the available video stream in your browser session, and gives you a cleaner way to save accessible content locally without using desktop recording tools.

Features
Video detection for direct Tella pages and supported embeds
In-page download controls on detected Tella players
Popup workflow for reviewing and downloading the current video
MP4 output for easier playback and transfer
Download support for screen recordings, presentations, demos, and walkthroughs
Browser-based workflow without desktop recording software
Progress tracking during active downloads
Cross-browser support for Chrome, Edge, Brave, Opera, Firefox, Whale, and Yandex
How It Works
Install the extension from the latest release.
Open a Tella page or a website with an embedded Tella video.
Start playback so the extension can detect the stream.
Use the on-page control or the extension popup.
Download the video and save the final MP4 file locally.
Step-by-Step Tutorial: How to Download Videos from Tella
Install TellaTV Downloader from the latest GitHub release.
Open the Tella page or supported embedded Tella player you want to save.
Let the player load fully and press play.
Click the extension button or the download control shown on the player.
Start the download once the video is detected.
Wait for the MP4 export to complete.
Open the saved file from your Downloads folder.

Troubleshooting

The extension is not detecting the video
Press play first and wait a few seconds so the stream has time to initialize.

The player button is missing
Open the extension popup directly. Some supported embeds work better through the popup UI.

The video is embedded on another website
Wait for the embed to finish loading before opening the extension.

The download did not start
Refresh the page, play the video again, and retry once the player is fully loaded.

The page requires access or login
The extension only works on videos you can already access in your current browser session.

Security & Scope
Operates only on the page the user intentionally opens in the active browser tab

FAQ

What kinds of Tella videos can I download?
Screen recordings, presentations, demos, and walkthrough videos hosted on Tella or shown through supported embedded Tella players.

Do I need to press play first?
Yes. Tella streams are typically only exposed after playback begins.

What file format do downloads use?
Videos are saved as MP4 files.

Does it work on embedded Tella players?
Yes. The extension supports Tella embeds on supported third-party pages.

Where are videos saved?
They are saved to your default Downloads location.

Do I need extra software?
No. Everything runs through the browser extension.
