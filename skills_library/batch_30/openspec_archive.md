/openspec-archive — Архивация изменения

Steps

Определи change-id для архивации:
Из аргументов $ARGUMENTS
Или посмотри openspec/changes/ и уточни у пользователя
Проверь что change завершён (все задачи в tasks.md отмечены [x])
Перемести openspec/changes/<id>/ в openspec/changes/archive/<id>/
Если есть spec deltas — обнови соответствующие спецификации в openspec/specs/
Валидируй результат

Reference

Проверь список изменений: ls openspec/changes/
Проверь спеки: ls openspec/specs/
